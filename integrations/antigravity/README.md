# Boss Mode for Antigravity

Antigravity natively supports the Boss Mode philosophy through its artifact system (`task.md` and `implementation_plan.md`).

To explicitly trigger this workflow, you can add this repository's workflows to your agent settings or use the provided workflow file.

## Setup
Ensure [boss-mode.md](../../.agents/workflows/boss-mode.md) is in your `.agents/workflows/` directory.

## Trigger
Simply tell Antigravity: "I want to start Boss Mode for [objective]" or "/boss-mode [objective]".
